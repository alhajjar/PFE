Bonjour

ce tuto vous montre les differentes etapes pour faire fonctionner le code PFE sur linux

les librairies necessaires sont:

1. Boost
2. Eigen3
3. Csparse
4. Tiff3

pour installer ces librairies sur Ubuntu vous ouvrez le software center et vous cherchez:
1. libboost-dev
2. libeigen3-dev
3. libtiff5-dev

apres l'installation de ces biblio vous ouvrez le terminal et vous navigez vers le dossier Csparse
vous faite: 
1. make clean
2. make

et vous entrez dans le dossier "lib" puis vous copiez libcsparse.a dans le dossier du projet (dans le meme emplacement du make file)

avant de faire un make du projet il vous faut le fichier libtiff.so 
il se trouve dans /usr/local/lib vous le copier dans le dossier du projet (PFElinux)
si vous ne le trouver pas vous telecharger tiff-4.0.2 du site:
"ftp://ftp.remotesensing.org/pub/libtiff"
et vous l'installez:
1. ./configure
2. make 
3. make install

puis vous navigez sur le terminal vers le dossier du PFE et vous faite:
1. make clean 
2. make
quand le make fini vous remarquez que dans le dossier "test" libpfe.a est cree
alors vous entrez dans le dossier test et vous compilez avec:
g++ -o output ihmlignecommande.cpp libpfe.a 

enfin vous executez l'output avec :
./output
