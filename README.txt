Mission :
	Réalisation d’une plateforme de simulation sur la
	multiplication-dispersion des pucerons

But : 
	Intégrer le composant logiciel réalisant la simulation
	numérique du modèle dans le Système d’Information (SI)
	d’ARVALIS.

Besoins :
	1.Ramener tout le logiciel et ses composants vers une des
	licences LGPL, BSD ou équivalent.

	2.Rendre le code portable entre Linux (basé debian, noyau
	≥ 2.6.32) et Windows XP/Seven.

	3.Créer une librairie dynamique (LD) en C++.
	
	4.Intégrer la LD dans le SI qui est lui-même écrit en Delphi et
	fonctionne sous Windows.
	
	5.Permettre la lecture/écriture de deux formats cartographiques (
	’ASCII Grid’ et ENVI) et de ﬁchiers CSV.
	
	6.Création d’un exécutable utilisable en ligne de commande sous
	Linux, basé sur la LD.

	7.Rédaction d’un manuel de l’utilisateur et d’un manuel du
	développeur
